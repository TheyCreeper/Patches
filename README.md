# Patches for AtlasOS
## How does a patch work?
A patch is a .zip file containing a powershell script that gets ran by AtlasOS Toolbox. Its a custom file made for each patch with the goal of applying bug fixes, adding new features, and making system improvements.

Each patch comes with a changelog.md file that describes the changes made in the update

It's the second step into seamless upgradability and migration torwards full installation via our custom powershell modules
